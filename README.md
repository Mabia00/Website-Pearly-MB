# Skincare Landing Page 🌿

Hydration Skincare Landing Page

## About the Project

The layout was based on a Figma design concept for a fictional vegan skincare brand.
The goal of the project was to practice front-end layout structure and responsive design.

## Technologies Used

- **HTML5**

- **CSS3**

- **Flexbox**

- **CSS Grid**

- **Responsive Design (Media Queries)**

## Features

- [x] Responsive layout for desktop, tablet, and mobile.

- [x] Hero section with product highlight.

- [x] Product cards section.

- [x] Customer reviews section.

- [x] Promotional banner.

- [x] Structured footer with navigation links.

## Responsiveness

The layout adapts to different screen sizes:

- Desktop.

- Tablet.

- Mobile.

- Media queries were used to reorganize elements and maintain usability across devices.
